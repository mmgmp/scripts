# scripts
Script varios

## menus
* [fzfcha](https://github.com/mmgmp/scripts/blob/main/menus/fzfcha) --> Menú fzf con imágenes.
* [fzfcha-bg](https://github.com/mmgmp/scripts/blob/main/menus/fzfcha-bg) --> Selector de fondo de pantalla.
* [tpass](https://github.com/mmgmp/scripts/blob/main/menus/tpass) --> Muestra las contraseñas de pass en un menu.
* [twsearch](https://github.com/mmgmp/scripts/blob/main/menus/twsearch) --> Buscar rapidamente en internet.

## notifications
* [screencontrol](https://github.com/mmgmp/scripts/blob/main/notifications/screencontrol) --> Notificación al cambiar el brillo de la pantalla.
* [volumecontrol](https://github.com/mmgmp/scripts/blob/main/notifications/volumecontrol) --> Notificación al cambiar el volumen.

## user-tools
* [cpant](https://github.com/mmgmp/scripts/blob/main/user-tools/cpant) --> Varios tipos de capturar la pantalla.
* [super-swaylock](https://github.com/mmgmp/scripts/blob/main/user-tools/super-swaylock) --> Swaylock con el fondo de pantalla.
* [tsession](https://github.com/mmgmp/scripts/blob/main/user-tools/tsession) --> Menú con varias opciones para la sesión.
